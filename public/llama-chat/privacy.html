プライバシーポリシー

1. 収集する情報

本アプリは、個人情報を外部サーバーへ送信しません。以下の情報を端末内に保存します。

  • 設定情報（OllamaサーバーURL、モデル名、各種パラメータ）
  • テンプレート名・キャラクター設定
  • チャット履歴
  • アバター画像ファイル

会話内容は、ユーザーが設定した Ollama 互換 API サーバーへ送信されます。開発者が会話データを収集することはありません。

2. 通信

  • 設定画面を開くと、モデル一覧更新のため設定済み Ollama サーバーの /api/tags へ GET リクエストを送信します（会話本文は含まれません）。
  • チャット時は、ユーザー入力（会話内容）を設定済み Ollama 互換 API サーバーへ送信します。
  • Web検索機能を有効にした場合、検索クエリは設定済み検索 API に送信されます。

3. 利用目的

上記データは、アプリの動作、チャット機能、表示、音声機能のために使用します。ユーザー入力は応答生成のためにのみ利用されます。

4. 音声機能

音声入力（音声認識）機能を使用する場合、端末の音声認識サービスを利用します。音声データの処理は Android システムに依存し、本アプリが独自に収集することはありません。

TTS（テキスト読み上げ）機能は端末内で処理されます。

5. 保存期間

設定・テンプレート・チャット履歴・アバター画像は、ユーザーが削除するかアプリのデータを消去するまで保持されます。チャット履歴は設定画面から削除できます。

Privacy Policy (English Version)

1. Information Collected

This application does not transmit any personal information to external servers. The following data is stored locally on the device:

  • Configuration data (Ollama server URL, model names, and parameters)
  • Template names and character settings
  • Chat history
  • Avatar image files

Conversation content is sent only to the Ollama-compatible API server configured by the user. The developer does not collect conversation data.

2. Communication

  • When opening settings, the app sends a GET request to /api/tags on your configured Ollama server to refresh model lists (no conversation content is included).
  • During chat, user input is sent to the configured Ollama-compatible API server.
  • When Web Search is enabled, search queries are sent to your configured search API.

3. Purpose of Use

The collected data is used solely for application functionality, chat features, display, and voice features. User input is used only for response generation.

4. Voice Features

Voice input uses your device's speech recognition service. Audio processing depends on Android system features, and this app does not independently collect voice data.

Text-to-speech is processed locally on the device.

5. Retention Period

Settings, templates, chat history, and avatar images are retained until the user deletes them or clears the application data. Chat history can be deleted from the settings screen.
