Llama Chat

AIを“使う”から、AIを“操る”へ。Llama Chatは、複数のAIモデルを自由に組み合わせ、研究者・開発者向けの実験機能を備えた高度なチャットプラットフォームです。

文書: 操作マニュアル | プライバシーポリシー

キャッチコピー例

  • 「AIが、あなたのために語り合う。」 — 複数のAIが自動で対話し、多角的な回答を導きます。
  • 「AIを“使う”から、AIを“操る”へ。」 — モデル選択やプロンプト制御で挙動を詳細に調整できます。
  • 「AI同士が語り合う。あなたは聞くだけ。」 — Chatterモードで2つのAIが自動で掛け合います。

主な特徴

  • Chatter モード（2 AI）: 2つのAIが交互に会話を続け、議論・翻訳・物語生成などを自動生成します。
  • 高度なカスタマイズ: モデル、システムプロンプト、音声設定、アバター（最大2体）をモデルごとに設定可能。
  • 音声ループ・Auto Voice Input: 応答後に自動で音声入力が再開することでハンズフリー体験を提供します。
  • Web Search 拡張: 任意の検索APIを接続して、生成コンテキストに検索結果を取り込めます。
  • 開発者向け機能: 詳細ログ、Debug Mode、APIリクエスト/レスポンスの可視化。
  • ローカル保存で高いプライバシー性: 設定・会話履歴・アバター画像は端末内に保存されます。
利用上の注意

Llama Chat は Ollama 互換の API サーバー（ローカルまたは同一ネットワーク）が必要です。Llama Tester の API 機能と組み合わせて利用することを想定しています。
