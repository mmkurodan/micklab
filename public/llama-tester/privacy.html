プライバシーポリシー

1. 収集する情報

本アプリは、個人情報を外部サーバーへ送信しません。以下の情報を端末内に保存します。

  • 設定情報（モデルURL、各種パラメータ、APIポート、ログレベル）
  • 構成ファイル（configs/*.json）
  • ダウンロードしたモデルファイル
  • ログファイル（ollama.log）

また、アプリの生成機能を利用する際、ユーザーが入力したテキスト（会話内容）は、端末内または同一ローカルネットワーク内で動作するローカルAPIに送信されます。このデータは外部サーバーへ送信されず、保存も行いません。

2. 通信

  • モデルのダウンロード時に、ユーザーが入力した URL へ通信します。
  • API サーバーを有効にした場合、端末内またはローカルネットワーク内のポート（0.0.0.0）でリクエストを受け付けます。
  • ユーザーの入力内容（会話全文）はローカル API に送信されますが、インターネットを経由して外部サーバーへ送信されることはありません。
  • ローカルネットワーク内からアクセス可能ですが、外部ネットワークからのアクセスは意図されておらず、アプリは外部公開を行いません。
  • ローカル通信は暗号化されていない場合がありますが、通信は端末内または同一ネットワーク内に限定されます。

3. 利用目的

上記データは、アプリの動作、生成機能、表示のために使用します。ユーザーの入力内容は生成処理のためにのみ利用され、外部送信・第三者提供・保存は行いません。

4. ログ

ログにはアプリの動作状況や API リクエスト情報が記録される場合があります。ログは端末内にのみ保存され、外部へ送信されません。

5. 保存期間

設定・モデル・ログは、ユーザーが削除するかアプリのデータを消去するまで保持されます。ログはメイン画面の「Clear Log」で削除できます。

Privacy Policy (English Version)

1. Information Collected

This application does not transmit any personal information to external servers. The following data is stored locally on the device:

  • Configuration data (model URL, parameters, API port, log level)
  • Configuration files (configs/*.json)
  • Downloaded model files
  • Log file (ollama.log)

When using the generation features, the text entered by the user (conversation content) is sent to a local API running on the device or within the same local network. This data is not transmitted to external servers and is not stored.

2. Communication

  • When downloading models, the application communicates with the URL entered by the user.
  • When the API server is enabled, it listens on a port (0.0.0.0) accessible within the device or the local network.
  • User input (full conversation text) is sent to the local API, but it is never transmitted over the internet or to any external server.
  • The API may be accessible from devices within the same local network, but it is not intended for external network access, and the application does not expose the API to the internet.
  • Local communication may be unencrypted, but all communication is restricted to the device or the same local network.

3. Purpose of Use

The collected data is used solely for application functionality, generation processing, and display. User input is used only for generation and is not transmitted externally, shared with third parties, or stored.

4. Logs

Logs may contain operational information or API request details. Logs are stored only on the device and are not transmitted externally.

5. Retention Period

Settings, models, and logs are retained until the user deletes them or clears the application data. Logs can be deleted using the "Clear Log" option on the main screen.
