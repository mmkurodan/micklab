日本語

1. 収集する情報

本アプリは、個人情報を外部サーバーへ送信しません。以下の情報を端末内に保存します。

  • 設定情報（モデル URL、各種パラメータ、API ポート、ログレベル、MCPコンフィグJSON、Function Definitions JSON）
  • 構成ファイル（configs/*.json）
  • ダウンロードまたは取り込んだモデルファイル
  • ログファイル（ollama.log、Javaクラッシュ時は last_crash.txt、ネイティブクラッシュ時は native_crash.txt）

また、アプリの生成機能、API連携、WebUIを利用する際、ユーザーが入力したテキスト（会話内容）は、端末内または同一ローカルネットワーク内で動作するローカル API に送信されます。MCPサーバーを設定して利用しない限り、このデータが外部サーバーへ送信されることはなく、保存も行いません。

2. 通信

  • モデルのダウンロード時に、ユーザーが入力した URL へ通信します。
  • ローカル .gguf ファイルの取り込み時は、端末内の選択ファイルをアプリ保存領域へコピーします。外部サーバー通信は行いません。
  • HTTPS URL を使用する場合は、通常のSSL/TLS証明書検証を行います。
  • API/WebUI サーバーを有効にした場合、端末内またはローカルネットワーク内のポート（0.0.0.0）でAPIとWebUIのリクエストを受け付けます。
  • ユーザーの入力内容（会話全文）はローカル API に送信されますが、インターネットを経由して外部サーバーへ送信されることはありません。
  • MCPサーバーを設定してメイン画面、API連携、またはWebUIから利用した場合、会話内容やツール入力の一部が設定先のMCPサーバーへ送信されることがあります。送信先や取扱いは各MCPサーバーの設定・運用に依存します。
  • ローカルネットワーク内からアクセス可能ですが、外部ネットワークからのアクセスは意図されておらず、アプリは外部公開を行いません。
  • ローカル通信は暗号化されていない場合がありますが、通信は端末内または同一ネットワーク内に限定されます。

3. 利用目的

上記データは、アプリの動作、生成機能、表示のために使用します。ユーザーの入力内容は生成処理のためにのみ利用され、外部送信・第三者提供・保存は行いません。

4. ログ

ログにはアプリの動作状況や API リクエスト情報が記録される場合があります。ログは端末内にのみ保存され、外部へ送信されません。

5. 保存期間

設定・モデル・ログは、ユーザーが削除するかアプリのデータを消去するまで保持されます。ログはメイン画面の「ログ消去」で削除できます。

English

1. Information Collected

This application does not transmit any personal information to external servers. The following data is stored locally on the device:

  • Configuration data (model URL, parameters, API port, log level, MCP config JSON, Function Definitions JSON)
  • Configuration files (configs/*.json)
  • Downloaded or imported model files
  • Log files (ollama.log, last_crash.txt for Java crashes, and native_crash.txt for native crashes)

When using the generation features, API integrations, or the WebUI, the text entered by the user (conversation content) is sent to a local API running on the device or within the same local network. Unless you configure and use MCP servers, this data is not transmitted to external servers and is not stored.

2. Communication

  • When downloading models, the application communicates with the URL entered by the user.
  • When importing a local .gguf file, the app copies the selected file into the app storage area and does not communicate with any external server.
  • For HTTPS URLs, standard SSL/TLS certificate verification is used.
  • When the API/WebUI server is enabled, it listens on a port (0.0.0.0) accessible within the device or the local network.
  • User input (full conversation text) is sent to the local API, but it is never transmitted over the internet or to any external server.
  • If you configure and use MCP servers from the main prompt input, API integrations, or the WebUI, parts of conversation content or tool inputs may be sent to those configured MCP servers. The destination and handling depend on each MCP server's own configuration and operation.
  • The API may be accessible from devices within the same local network, but it is not intended for external network access, and the application does not expose the API to the internet.
  • Local communication may be unencrypted, but all communication is restricted to the device or the same local network.

3. Purpose of Use

The collected data is used solely for application functionality, generation processing, and display. User input is used only for generation and is not transmitted externally, shared with third parties, or stored.

4. Logs

Logs may contain operational information or API request details. Logs are stored only on the device and are not transmitted externally.

5. Retention Period

Settings, models, and logs are retained until the user deletes them or clears the application data. Logs can be deleted using the "Clear Log" option on the main screen.
