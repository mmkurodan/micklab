操作マニュアル

1. アプリ概要

本アプリは端末上でLLMを実行し、プロンプトに対する回答を生成します。必要に応じてOllama互換APIサーバーを起動できます。

2. 初期設定（推奨手順）

  1. メイン画面で「Settings」を開きます。
  2. モデルURLを入力し「Load Model」を押してモデルを読み込みます。
  3. パラメータを調整する場合は各項目を編集し「Save Config」で保存します。

3. メイン画面

  • Enter Prompt: プロンプトを入力します。
  • Send: 生成を開始します（モデル未読み込み時はエラー表示）。
  • Re-init Model: 現在のモデルを解放して再初期化します。
  • View Log: ログファイル内容を出力欄に表示します。
  • Clear Log: ログファイルを空にします。
  • Start/Stop API Server: APIサーバーを起動/停止します。
  • Copy: 出力/ログをクリップボードへコピーします。

4. 設定画面の操作

  • Configuration Management: 設定の保存/削除/読み込みを行います。
  • Model Selection: モデルURLを指定し読み込みます。
  • Model Parameters: 生成パラメータを設定します。
  • Prompt Template: {USER_INPUT} で入力を差し込みます。
  • Llama API Server: サーバーポートを指定します。
  • Log Settings: ログレベルを選択します。
  • Show License: ライセンス文面を表示します。
  • Documents: 権利、操作マニュアル、プライバシーポリシーを確認できます。

5. APIサーバー（任意）

  • 起動すると端末内で /api/chat, /api/generate, /api/tags を提供します。
  • 同時生成は1件のみです（ビジー時は503を返します）。
  • Android 13以上では通知権限が必要な場合があります。
