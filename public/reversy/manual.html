操作マニュアル

このページは、現在の Android 実装に基づく Dual AI Reversy の操作ガイドです。アプリ内ドキュメントの内容をベースに、公開ページ向けに整理して掲載しています。

関連文書: プライバシーポリシー

1. 対局準備

  • 画面上部の 黒 / 白 ボタンで、それぞれの操作モードを ユーザ / CPU / AI の順に切り替えます。
  • 黒設定 / 白設定 では、CPU 深さ、AI 接続先 URL、モデル、AI プロンプト、タイムアウト、再試行回数を調整できます。
  • AI モードでは api/tags からモデル一覧を取得して選択できます。

2. 対局開始と進行

  • 対局開始 を押すと新しい対局を始めます。
  • 対局が始まると同じ位置のボタンが リセット に変わり、盤面を初期配置へ戻せます。
  • ユーザ操作の手番では、盤面の置けるマスをタップして石を置きます。
  • CPU や AI を含む対局でも、必要になった時点でリセットできます。

3. コメント欄

  • 画面下部のコメント欄には、CPU や AI の着手理由が新しい順で表示されます。
  • コメント本文はスクロールとコピー選択ができます。
  • まだ自動対戦コメントが無い場合は案内文が表示されます。

4. AI モード

  • AI モードでは、設定した URL の API サーバーへ盤面情報、合法手候補、AI 用プロンプトを送信して着手候補を取得します。
  • 応答は 1 行目に座標、2 行目に理由を返す形式を前提としています。
  • 設定画面の 黒AIログ / 白AIログ から送受信履歴を確認・コピー・クリアできます。
  • 既定の接続先は http://127.0.0.1:11434 です。

5. CPU 思考

  • CPU は角の確保、相手の合法手数、自分の将来の手数、相手が角を取り返せるかなどを考慮して着手を選びます。
  • CPU深さ は設定画面から 1〜10 の範囲で切り替えられます。

6. ドキュメント

  • ドキュメント ボタンから、操作マニュアル、Privacy Policy、権利表記を参照できます。
  • ドキュメント画面では、選択中の文書をクリップボードへコピーできます。

7. 権利表記の要点

  • ゲームロジック、UI、説明文、対局コメント表示などの独自実装部分の権利は本アプリ提供者に帰属します。
  • 同梱アイコン画像やアバター画像などのアセットは、本アプリの配布物として管理されています。
  • AI モードで接続するモデル、API、サーバー応答の権利は、それぞれの提供元に帰属します。

8. 注意事項

  • 外部 AI サーバーを使う場合は、送信先 URL の安全性と利用規約を確認してください。
  • 通信できない場合や応答形式が想定と異なる場合、AI の着手は失敗することがあります。
