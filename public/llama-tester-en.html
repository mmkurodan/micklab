LLM tester with llama.cpp

LLM tester with llama.cpp is a local LLM testing tool for Android that lets you load GGUF models, tune inference settings, inspect prompt templates and logs, and optionally expose an Ollama-compatible API from a single app.

Docs: User Manual | Privacy Policy

  • Supports both model downloads from a URL and importing local .gguf files from the device.
  • Lets you save and reuse profiles for generation settings, Think behavior, custom chat templates, and display language.
  • Can start an on-device Ollama-compatible API server so other clients can call endpoints such as /api/chat.

Screenshots

Key Features

  • On-device local inference: Runs GGUF models directly on Android with llama.cpp.
  • Flexible model loading: Supports downloadable model URLs and local .gguf imports.
  • Deep inference controls: Adjust n_ctx, n_threads, GPU Offload Layers, Top-p, Top-k, penalties, Mirostat, DRY, Think behavior, and custom chat templates.
  • Configuration profiles: Save, load, and delete profiles, then apply them immediately with SAVE & CLOSE.
  • Practical log/output workflow: View the latest 100 log lines, refresh logs, clear logs, copy results, and save responses or logs.
  • Built-in Ollama-compatible API: Provides /api/chat, /api/generate, and /api/tags; only one generation runs at a time, with a queue of up to 10 requests for up to 60 seconds.

Operational Notes

  • Model downloads can be several gigabytes. Wi‑Fi is strongly recommended.
  • The local API server is intended for the same device or a local network. Android 13+ may require notification permission.
  • Generation speed and stability depend on model size, device memory, thread settings, and GPU offload configuration.
