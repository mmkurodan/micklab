Android / llama.cpp / GGUF

LLM tester with llama.cpp

LLM tester with llama.cpp は、Android 端末上で GGUF モデルを読み込み、推論設定・プロンプトテンプレート・ログ・Ollama 互換 API を一つのアプリで扱えるローカルLLM検証ツールです。

文書: 操作マニュアル | プライバシーポリシー

  • モデル URL からのダウンロードと、端末内 .gguf ファイルの取り込みに対応。
  • 生成パラメータ、Think 設定、カスタム chat template、表示言語をプロファイルとして保存・再利用できます。
  • 必要に応じて端末内で Ollama 互換 API サーバーを起動し、他のクライアントから /api/chat などを利用できます。

スクリーンショット

主な特徴

  • 端末内でのローカル推論: llama.cpp を使って Android 端末上で GGUF モデルを直接実行します。
  • モデル導入の自由度: HTTP/HTTPS のモデル URL を指定した読み込みと、ローカル端末からの .gguf 取り込みの両方に対応しています。
  • 詳細な推論制御: n_ctx、n_threads、GPU Offload Layers、Top-p、Top-k、Penalty、Mirostat、DRY、Think、custom chat template まで調整できます。
  • 構成プロファイル管理: 設定の保存・読み込み・削除ができ、SAVE & CLOSE で現在のモデルへ即時反映されます。
  • ログと出力の扱いやすさ: 最新 100 行のログ表示、ログ更新、ログ消去、応答/ログのコピーと保存に対応しています。
  • 内蔵 Ollama 互換 API: /api/chat、/api/generate、/api/tags を提供し、同時生成は 1 件、待機キューは最大 10 件・最大 60 秒です。

運用メモ

  • 大きなモデルの取得には数 GB 単位の通信が必要になる場合があります。モデルのダウンロードは Wi-Fi 環境が推奨です。
  • ローカル API サーバーは同一端末またはローカルネットワーク向けの利用を想定しています。Android 13 以降では通知権限が必要になる場合があります。
  • 生成速度や安定性は、モデルサイズ・端末メモリ・スレッド設定・GPU オフロード設定に依存します。
